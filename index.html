📋 Changelog — Spartans Manager
v2.5 · Maggio 2026
Nuove funzionalità
Multi-account Google — ogni utente accede con il proprio account Google personale; l'admin crea il file dati e condivide il File ID; gli altri utenti incollano il File ID nelle Impostazioni e accedono in lettura/scrittura senza condividere credenziali
Layout responsive completo — su tablet e telefono la sidebar diventa un menu a tendina (bottone ☰); i form vanno su colonna singola; le tabelle scorrono orizzontalmente; i modali si aprono dal basso su mobile
PWA (Progressive Web App) — aggiunto manifest.json; su Android/iOS appare "Aggiungi alla schermata home"; si installa come app nativa senza barra del browser a schermo intero
Fasi sensibili Martin 1982 corrette — la tab non è più una valutazione ma una guida per il tecnico: mostra quali capacità sono in fase ottimale (🔴 Alta priorità), prossima (🟡 Media) o fuori fase (⚪ Bassa) per quell'atleta in questo momento, con finestra biologica ±2 anni
Modifiche
Scope OAuth aggiornato da drive.appdata a drive.file per supportare file condivisibili
Login con prompt select_account — ogni utente può scegliere il proprio account Google
Impostazioni: aggiunta sezione File ID condiviso con copia per admin e incolla per utenti
Backup Drive spostato su file separato nella cartella Drive dell'admin
v2.4 · Maggio 2026
Nuove funzionalità
Sezione Comunicazioni — seleziona destinatari per corso/tipologia, componi messaggio con oggetto e firma; invio via email (BCC), copia testo per WhatsApp Broadcast List, esportazione contatti .vcf; slot riservato per integrazione futura WhatsApp Business API
Sezione Valutazione atleti — scheda completa per ogni atleta con 3 sezioni: Psico-fisica (capacità fisiche + tratti psicologici: aggressività, leadership, spirito di squadra), Tecnica (pattinaggio, bastone, passaggio, tiro, finte — dai moduli A/B/C/D FISR), Tattica (individuale offensiva/difensiva + collettiva — dalle tabelle FISR pag. 224-236)
Fasi sensibili Martin 1982 — tab dedicato nella valutazione: visualizza per ciascuna capacità le fasce d'età sensibili (M/F) con evidenziazione della finestra biologica dell'atleta (±2 anni rispetto all'età anagrafica)
Modulo H2 FISR — generazione automatica del modulo elenco atleti per gara, compilato con i dati delle convocazioni; accessibile dal dettaglio evento con pulsante "📋 Modulo H2"; stampabile/salvabile come PDF
Modifiche
Sidebar: aggiunta voci Comunicazioni (📨) e Valutazione atleti (📊)
Dettaglio evento gara: aggiunto pulsante Modulo H2 accanto a Convocazioni
DB: aggiunto campo valutazioni per ogni atleta
v2.3 · Maggio 2026
Nuove funzionalità
Storico presenze filtrato per mese e anno — selettori mese e anno nello storico presenze per visualizzare singoli periodi
Ordinamento crescente date presenze — lo storico è ora ordinato dal più vecchio al più recente
Riepilogo statistico storico — numero sessioni, media presenti e percentuale per ogni data
Età anagrafica automatica — campo calcolato dalla data di nascita, non modificabile, visibile e ordinabile in tabella anagrafiche
Categoria FISR automatica — calcolata dinamicamente in base all'anno di nascita e alla stagione sportiva attiva, secondo le norme FISR Hockey Inline (CU n.044/2022): Primi Passi, Minihockey, Under 10, Under 12, Under 14, Under 16, Elite, Under 20, Seniores, Amatori
Filtro per categoria FISR — il selettore "Tutte le categorie" usa le categorie ufficiali FISR, con calcolo automatico che si aggiorna ad ogni nuova stagione
Modifiche
Tabella anagrafiche: aggiunte colonne Età e Categoria FISR
Storico presenze: aggiunto riepilogo con media e percentuale presenze
